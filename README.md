Los Primeros 12 – PWA
A mobile-first Progressive Web App focused on infant development guidance for new parents.

This project was created as part of the PandA brand ecosystem, combining emotional design, accessibility, and offline-first web technologies.

Features
Mobile-first responsive design
Installable Progressive Web App (PWA)
Offline support with Service Workers
Lightweight and fast loading
Clean and family-oriented UI
Custom branding and icons
Tech Stack
HTML5
CSS3
JavaScript
Service Workers
